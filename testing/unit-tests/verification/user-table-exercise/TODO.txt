Exercise: Builder and Custom Assertions - User Table
-------------------------------------------------------------------------------

Given the classes "User" and "UserTable" als well as "UserTableTest".

A) Builder 
    In user_table_builder.py, implement a "UserTableBuilder" class which 
    can be used to create a "UserTable" instance together with "User" 
    instances.
    
    Example: UserTableBuilder()
                .user(3,"Lisa","lKi8/hhgT")
                .user(7,"Bart","kJ7&fRsd34%7")
                .build()

B) Custom Assertions
    In tesr_user_table.py, implement a custom assertion method for 
    the "User" class so that we can compare "User" objects in one line.
    
    Example: self.assertEqualUser(expected:User, actual:User)
