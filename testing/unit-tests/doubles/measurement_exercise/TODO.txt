Exercise: Mock Object + Code Coverage
-------------------------------------------------------------------------------

Given the implementation of the "Multimeter" class, implement all test cases
needed to realize a code coverage of 100%.

A) Device - Protocol
Define a "Device" Protocol in the "measurement.py" file which can be used 
as a specification for a Mock object.
Also add "Device" type hints to the "Multimeter" class. 

B) Test Cases  
Implement all test cases, using Mock objects for "Multimeter" in the file called 
"test_multimeter.py", to reach a 100% code coverage.

Tip: Use the following command to measure the code coverage:
$ pytest test_multimeter.py --cov=data_analysis --cov-report=term-missing
