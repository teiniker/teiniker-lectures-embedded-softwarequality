Exercise: Mock Object + Code Coverage
-------------------------------------------------------------------------------

Given the implementation of the "DataAnalysisService" class, implement all test 
cases needed to realize a code coverage of 100%.

A) DataAccess - Protocol
Define a "DataAccess" Protocol in the "data_analysis.py" file which can be used 
as a specification for a Mock object.

B) Test Cases
Implement all test cases, using Mock objects for "DataAccess" in the file called 
"test_data_analysis.py", to reach a 100% code coverage.


Tip: Use the following command to measure the code coverage:

$ pytest test_data_analysis.py --cov=data_analysis --cov-report=term-missing
