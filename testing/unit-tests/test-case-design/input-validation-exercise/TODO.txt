Exercise: Test Case Design based on Code Coverage - Input Validation
-------------------------------------------------------------------------------

To test the "User" class defined in "user.py", implement a comprehensive set 
of unit test cases that achieves 100% code coverage.

Put all tests in a file called "test_input_validation.py".

Use the following commands:

$ pytest test_input_validation.py --cov=user --cov-branch

$ pytest test_input_validation.py --cov=user --cov-branch --cov-report=html
