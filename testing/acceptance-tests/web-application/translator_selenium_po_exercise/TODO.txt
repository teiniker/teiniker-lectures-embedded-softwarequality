Exercise: Selenium Tests Using the Page Object Pattern - Translator
-------------------------------------------------------------------------------

Given working Selenium tests for the Translator web application:

To start the web application, open a new terminal and type:
$ cd translator/
$ java -jar translator.jar

A) Page Object Classes 
Use the Selenium WebDriver tests implemented in "test_translator_page_object.py"
and add the following page object classes at the beginning of this file:

    class TranslatorPO
        + word 
        + language 
        + translate()

    class TranslatorResultPO
        + message

B) Refactoring Test Cases 
Refactor the given test cases to make them work with the implemented page objects.
